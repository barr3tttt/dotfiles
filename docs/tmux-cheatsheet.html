Terminal Cheatsheet - tmux / WezTerm / Neovim

tmux prefix is Ctrl+a: tap it, release, then the command key. "prefix c" = Ctrl+a, then c.
Yellow keys = the prefix (Ctrl+a).   The big idea: a session holds windows (tabs), a window holds panes (splits). Detach and your work keeps running on a background server.

The mental model

Session = a whole workspace (survives, reattachable) contains → Window = a tab (one full screen) contains → Pane = a split inside a window
tmux runs inside WezTerm and owns all splitting. WezTerm just gives you the window, tabs, and font controls.

Sessions (the superpower)

  • tmuxstart tmux
  • tmux new -s workstart a named session
  • prefixthenddetach (work keeps running)
  • tmux attachreattach to it
  • tmux lslist sessions
Detach, close the terminal, ssh back later, attach - everything is exactly as you left it.

Windows (tabs)

  • prefixcnew window
  • prefixn / pnext / previous
  • prefix0..9jump to window N
  • prefix,rename window
  • prefix&close window

Panes (splits)

  • prefix|split left / right
  • prefix-split top / bottom
  • prefixzzoom pane fullscreen (toggle)
  • prefixHJKLresize (hold Shift, repeat)
  • prefixxclose pane

Move between panes (no prefix!)

  • Ctrl+hjklmove left / down / up / right
vim-tmux-navigator: the same keys move across Neovim splits AND tmux panes seamlessly. No prefix, no thinking about which is which.

Copy / scroll mode

  • prefix[enter scroll mode
  • vstart selecting (vi-style)
  • yyank to clipboard + exit
  • qquit scroll mode
Mouse is on too: scroll, click panes, drag borders.

Gotchas + housekeeping

  • Ctrl+aCtrl+asend a real Ctrl+a to the shell
  • prefixCtrl+lclear screen (bare Ctrl+l now navigates)
  • prefixrreload tmux config

WezTerm (the outer window)

  • Ctrl+Shift+t / wnew / close tab
  • Ctrl+= / - / 0font bigger / smaller / reset
Splitting is tmux's job, so WezTerm has no split keys. Save wezterm.lua and it hot-reloads live.

Neovim (leader = Space)

  • Spaceffuzzy find files (project root)
  • Spacesgrep across the project
  • Ctrl+hjklmove across splits + tmux panes
  • Spaceopen which-key menu (everything else)

First session - 2 minutes to muscle memory

  • Type tmux and press Enter. One session, one window, one pane.
  • prefix- then prefix| to make a few panes. Run ls and top in them.
  • Ctrl+h/j/k/l to hop between panes. No prefix.
  • prefixz to fullscreen one pane, then again to bring the others back.
  • prefixc for a new window; prefixn/p to flip between windows.
  • prefixd to detach, then tmux attach - your whole workspace is back. That is the whole point of tmux.
